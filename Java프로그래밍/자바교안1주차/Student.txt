교육생의 교육비에 따른 환급금을 계산하는
클래스를 작성하세요.

1. 교육생 클래스에는 
<변수>
이름, 소속("멀티캠퍼스" 공유), 과정명, 교육비, 부가금(입력받아 전달 이후 수정 불가), 환급금을 
나타내는 변수를 정의합니다.
교육비 부가금 환급금 - 산술 연산 가능한 타입 
이름 소속 과정명 - 문자열 타입

  
2. main(String[] args) 메서드를 포함하는 또 다른 클래스를 
작성합니다.
 2-1> 명령행 매개변수를 이용하여 사용자로부터
  student가 참조하는 객체의 이름, 소속, 과정명, 
  교육비, 부가금을 입력받습니다.

 실행 : 
run-run configurations - -arguments-program arguments 차례대로 선택
이자바 JSP프로그래밍 50000 5000

2-2> 교육생 클래스의 객체를 생성하여 student
 라는 이름의 변수로 참조하도록 합니다.
이 때 2-1 입력 5개 변수를 이름, 소속, 과정명,
교육비, 부가금 변수값으로 초기화합니다.

2-3>  환금급을 계산하자

  - 환급금은 
   과정명이 "자바프로그래밍" 이면
   입력한 교육비의 25% + 부가금 이고,
   
   과정명이 "JDBC프로그래밍" 이면
   입력한 교육비의 20% + 부가금 이고,
   
   과정명이 "JSP프로그래밍" 이면
   입력한 교육비의 15% + 부가금 으로        
   계산하여 초기화합니다.

   그밖의 과정명은 
   "잘못 입력되었습니다"는 에러 메시지를 출력합니다. 메소드 중단합니다.

 - 중단되지 않았다면, student 내의 
이름, 소속, 과정명, 교육비, 부가금, 환급금을 출력합니다.
